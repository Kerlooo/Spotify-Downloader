# Spotify Downloader
# Requirements
- Python 3.7 >=
- spotdl

# How to use it
## Debian
```bash
git clone https://github.com/Kerlooo/Spotify-Downloader.git
cd Spotify-Downloader
source myenv/bin/activate
python3 downloader.py
```

## Windows 
```bash
git clone https://github.com/Kerlooo/Spotify-Downloader.git
cd Spotify-Downloader
pip install spotdl
python3 downloader.py
```
